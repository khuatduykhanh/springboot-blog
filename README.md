# Đăng nhập 
Hệ thống sẽ trả về accessToken và refreshToken. AccessToken sẽ có quyền truy cập trong khoảng 604800000 ms. Sau khi hết thời gian accessToken sẽ hết hiệu lực.
RefeshToken sẽ giúp hệ thống tự trả lại accessToken nếu token đã đã hết hạn mà không cần đăng nhập lại, refreshtoken sẽ hết hạn sau 86400000 ms. Sau khoảng thời gian này hệ thống sẽ bắt bạn đăng nhâp lại
